# Third-Party Notes

`Reach Crafting` is being developed with the following reference projects in mind:

- `OneClickCrafting` by BreadMoirai
  Repository: `https://github.com/BreadMoirai/OneClickCrafting`
  License: MIT
- `stack-to-nearby-chests` by xiaocihua
  Repository: `https://github.com/xiaocihua/stack-to-nearby-chests`
  License: LGPL-3.0-or-later

If code is copied or adapted from those projects, their original notices and license obligations must be preserved.

Practical rule of thumb:

- Using the projects as references for ideas and behavior does not by itself copy their code.
- Copying source, adapting source closely, or carrying over nontrivial original code does trigger license obligations.
- For MIT-derived portions, preserve the original copyright and permission notice.
- For LGPL-derived portions, preserve notices, mark changes where appropriate, and keep distribution terms compatible with LGPL obligations.
